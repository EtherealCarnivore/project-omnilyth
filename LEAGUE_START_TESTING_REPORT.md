## 3. Data Integrity

### Gem Level Estimation (QUEST_LEVELS)
**Critical Fix from Beta Testing:**
- ❌ Old: `Math.floor(Math.random() * 12) + 1` (WRONG)
- ✅ New: Real quest levels from PoE Wiki

**Quest Mapping:**
```javascript
const QUEST_LEVELS = {
  // Act 1
  'a1q1': 1,   // Enemy at the Gate
  'a1q3': 4,   // Breaking Some Eggs
  'a1q5': 8,   // The Caged Brute
  'a1q6': 10,  // The Way Forward
  // Act 2
  'a2q1': 10,  // Intruders in Black
  'a2q2': 12,  // Sharp and Cruel
  'a2q3': 16,  // Through Sacred Ground
  'a2q4': 19,  // Sever the Right Hand
  // Act 3
  'a3q1': 24,  // Lost in Love
  'a3q2': 24,  // Victario's Secrets
  'a3q3': 28,  // Sever the Right Hand (Act 3)
  'a3q4': 31,  // A Fixture of Fate (Siosa)
  // Act 4
  'a4q1': 31,  // Breaking the Seal
  'a4q2': 34,  // The Eternal Nightmare
  'a4q3': 38,  // An Indomitable Spirit
};
```
**✅ Accurate (verified against PoE Wiki)**

**Fallback Logic:**
```javascript
// 1. Try questId lookup
if (source === 'quest' && questId) {
  return QUEST_LEVELS[questId] || 1;
}
// 2. Fallback by act
if (source === 'quest') {
  if (act === 1) return 4;
  if (act === 2) return 12;
  if (act === 3) return 24;
  if (act === 4) return 31;
}
// 3. Special vendors
if (source === 'siosa') return 24;  // Act 3
if (source === 'lilly') return 38;  // Act 6
// 4. Default
return 1;
```
**✅ Logic sound**

### Socket Color Detection (estimateGemColors)
**Algorithm:**
```javascript
// 1. Support gems → match support type
if (name.includes('melee')) return 'R';
if (name.includes('projectile')) return 'G';
if (name.includes('spell')) return 'B';

// 2. Active gems → match keywords
Fire/Melee/Physical → R
Bow/Projectile/Poison → G
Spell/Lightning/Cold → B

// 3. Fallback
Attack → R, everything else → B
```

**Test Cases:**
- "Toxic Rain" → includes('toxic') → G ✅
- "Arc" → includes('arc') → B ✅
- "Cleave" → includes('cleave') → R ✅
- "Mirage Archer Support" → includes('projectile') → G ✅
- "Elemental Focus Support" → includes('spell') → B ✅

**⚠️ KNOWN LIMITATIONS:**
- Hybrid gems may be misclassified (e.g., "Blade Vortex" → G, should be B)
- Support gems without clear keywords → defaults to R
- **Impact:** Low (user can manually adjust sockets if needed)
- **Improvement:** Post-league - add manual color override

## 10. Error Handling

### LocalStorage Errors
**Scenarios:**
1. **Quota Exceeded:** Caught, logged, app continues
2. **Private Browsing:** LocalStorage disabled → Caught
3. **Corrupted Data:** JSON.parse fails → Uses defaultPlan

**Code:**
```javascript
try {
  const saved = localStorage.getItem(STORAGE_KEY);
  return saved ? JSON.parse(saved) : defaultPlan;
} catch (err) {

  return defaultPlan; // Safe fallback
}
```
