# PoE 2 baseline — pre-0.5

This file captures Path of Exile 2's current state as of patch **0.4 ("Last of the Druids")**, immediately before the 0.5 announcement (2026-05-07) and launch (2026-05-29). The user's directive: **0.5 is Omnilyth's first PoE 2 league.** This baseline is reference context, not implementation target — content tied to 0.4 is out of scope for Omnilyth.

## Game shape (0.4)

```
Campaign         : 6 acts (vs PoE 1's 10)
Endgame          : Atlas of Worlds-style passive tree (~8 meaningful nodes —
                   "sucks" per GGG; expanding to 40+ in 0.5)
Map equivalent   : Waystones (consumed to open maps, not stacked in stash like
                   PoE 1 maps; mod pool different)
Currency reset   : Per-league (last reset: Last of the Druids start, 2026-01-?)
Player base      : Smaller than PoE 1 but growing; Steam concurrents cycle
                   ~50-150k for league launches vs PoE 1's ~150-200k
```

## Mechanics that exist in PoE 2 (0.4)

| Mechanic | PoE 1 analog | Notes |
|---|---|---|
| Skill gems | Yes | But the system differs: **uncut gems** (raw drops) → **skill gems** + **spirit gems** + **support gems** + **meta-gems** (cast on X) |
| Passive tree | Yes | Smaller tree; different node taxonomy; orbit-based geometry similar enough to PoE 1's |
| Ascendancies | Yes | 3 ascendancies per class (vs PoE 1's typically 3 per class); unlocked via Trial of Sekhemas / Trial of Chaos |
| Item mods | Yes | Different mod pool; different generation rules; "items cannot be fully reset" — a fundamental design pivot |
| Crafting | Yes (different) | Desecrating replaces unveiling; Omens ≈ meta-crafting; talismans, corrupted mods, socketables. **Craft of Exile** has a PoE 2 mode that owns the deep math |
| Atlas tree | Yes (small) | ~8 meaningful nodes pre-0.5; not really worth planning until 0.5 |
| Trade | Yes | Official `pathofexile.com/trade2/` (different URL from PoE 1's `/trade/`); same core flow |
| Stash search | Yes | Verify whether regex limit / regex support is identical to PoE 1's 250-char cap — affects whether item-regex / gem-regex tools fork cleanly |

## Gem system (PoE 2 0.4 specifics)

PoE 2 distinguishes:
- **Uncut Skill Gems** (drops from monsters; consumed to "create" a chosen skill gem)
- **Skill Gems** (red/green/blue/white still — but not socketed into gear; equipped directly)
- **Spirit Gems** (passive aura-like effects, Reservation cost)
- **Support Gems** (linked to specific skill gems, max 5 per skill or 6 with quality)
- **Meta-Gems** (cast-on-X triggers — Cast on Critical Strike, Cast on Stunned, etc.)

Implication for Gem Browser / Gem Regex / Gem Planner forks: the data shape is different and the UI surface needs game-aware filters. The PoE 1 gem-availability-by-act data **does not carry over** — vendors are different, quest rewards are different.

## Atlas tree state (0.4 → 0.5)

Pre-0.5: ~8 meaningful nodes; not worth planning.

Post-0.5 (announced): expanding to 40+ nodes; framed by GGG as "second layer of build crafting." This is **the** unlock that makes Atlas Tree Planner PoE 2 viable as a tool.

## Waystones

PoE 2's endgame map item:
- Tier 1–15 (vs PoE 1's T1–T17).
- Mod pool different from PoE 1 maps.
- Consumed (not held in stash like a typical PoE 1 map collection).
- 0.5 announced an overhaul; expect mod pool to change materially.
